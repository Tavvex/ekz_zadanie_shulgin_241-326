# Электронная библиотека

Автоматизированная информационная система «Электронная библиотека» — курсовой проект
(Вариант 1: функциональность модерации рецензий).

Стек: **Flask**, **Flask-SQLAlchemy**, **Flask-Login**, **Flask-WTF**, **Bleach**, **Markdown**, **EasyMDE**, **Bootstrap 5**.

## Структура проекта

```
app/
  auth/        — аутентификация (вход/выход), декораторы доступа
  main/        — главная страница со списком книг
  books/       — просмотр/добавление/редактирование/удаление книг
  reviews/     — рецензии: создание, "Мои рецензии", модерация
  templates/   — шаблоны Jinja2
  static/      — статика (CSS, обложки, демо-изображения)
  models.py    — модели SQLAlchemy
  forms.py     — формы Flask-WTF
  utils.py     — Markdown / санитайзер (Bleach)
  cli.py       — команды flask init-db / seed-db
config.py
run.py
requirements.txt
```

## Установка и запуск

1. Создать и активировать виртуальное окружение:

   ```
   python -m venv venv
   venv\Scripts\activate
   ```

2. Установить зависимости:

   ```
   pip install -r requirements.txt
   ```

3. Создать таблицы в БД и заполнить тестовыми данными:

   ```
   flask --app run init-db
   flask --app run seed-db
   ```

   По умолчанию используется SQLite (файл `library.db`). Чтобы использовать MySQL,
   задайте переменную окружения `DATABASE_URL`, например:

   ```
   set DATABASE_URL=mysql+pymysql://user:password@localhost/library
   ```

4. Запустить сервер:

   ```
   flask --app run run --debug
   ```

   Приложение будет доступно по адресу http://127.0.0.1:5000/.

## Реализованная функциональность

- Каталог книг с пагинацией (10 книг на странице), сортировкой по году, жанрами,
  средней оценкой и количеством рецензий.
- Доступ к функциям в зависимости от роли (просмотр / редактирование / удаление /
  добавление книги).
- Добавление и редактирование книги через общий шаблон формы; загрузка обложки
  с дедупликацией по MD5-хэшу; описание в Markdown (EasyMDE) с санитайзингом
  через Bleach.
- Удаление книги с подтверждением в модальном окне и каскадным удалением
  связанных записей (обложка, рецензии, связи с жанрами) и файла обложки.
- Просмотр книги: обложка, данные, описание (Markdown → HTML), список одобренных
  рецензий.
- Создание рецензии (оценка 0–5, текст с поддержкой Markdown), статус
  «На рассмотрении» по умолчанию.
- «Мои рецензии» — список рецензий пользователя со статусами.
- «Модерация рецензий» — список рецензий на рассмотрении с пагинацией,
  страница рассмотрения с кнопками «Одобрить» / «Отклонить».
